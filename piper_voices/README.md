# Piper voice models

The `.onnx.json` config files are in the repo. The `.onnx` model files are large (~60 MB each) and are not tracked.

**On a fresh clone:** download the voice model(s) you need and place them here next to the matching `.onnx.json`. For example, from [rhasspy/piper-voices](https://huggingface.co/rhasspy/piper-voices) on Hugging Face. Custom voices (e.g. Arnold) need to be built or sourced separately and placed in this folder.
